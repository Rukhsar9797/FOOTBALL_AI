>>Algorithms Used
Object Detection: YOLOv11 Object Detection
Multi-Object Tracking: ByteTrack Algorithm
Player ID Assignment: Object Tracking with Persistent Track IDs
Jersey Color Extraction: HSV Color Space Analysis
Team Classification: K-Means Clustering for Color Classification
Player Distance Calculation: Euclidean Distance Algorithm
Player Speed Estimation: Velocity Calculation (Distance / Time)
Ball Possession Estimation: Nearest Neighbor Distance Algorithm
Player Heatmap Generation: Spatial Density Mapping with Gaussian Smoothing
Ball Heatmap Generation: Coordinate Density Mapping
Ball Trajectory Tracking: Coordinate Path Tracking Algorithm
Player Activity Score: Weighted Scoring Algorithm
Momentum Analysis: Field Zone-Based Analysis
Video Frame Processing: OpenCV Frame Processing
Data Handling: JSON-Based Data Processing
Visualization: Streamlit Dashboard
