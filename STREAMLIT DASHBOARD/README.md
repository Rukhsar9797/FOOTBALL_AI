## 📸 Features

| Page | Description |
|------|-------------|
| 🏠 **Home** | KPI cards, possession bar, system architecture overview |
| 📡 **Live Match** | Tracked video player + frame-by-frame inspector with detection overlays |
| 📊 **Team Analysis** | Possession donut, head-to-head comparison charts, speed distributions |
| 👤 **Player Analysis** | Searchable roster, radar performance profile, movement trail, speed timeline |
| 🌡️ **Heatmaps** | Zoomable White Team / Red Team positional heatmap + ball heatmap, overlaid on pitch outline |
| 💡 **Match Insights** | Player awards, auto-generated insights, leaderboards |
| ⬇️ **Downloads** | One-click export of video, analytics JSON, and detections JSON |
| ℹ️ **About** | Team info, system architecture diagram, tech stack |


## 🛠️ Tech Stack

- **Python 3.11+**
- **Streamlit** — Dashboard framework
- **Plotly** — Interactive charts
- **Pandas** — Data manipulation
- **NumPy** — Numerical operations
- **OpenCV** — Video frame extraction
- **Pillow** — Image loading & heatmap display
